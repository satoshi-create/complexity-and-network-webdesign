## ✅ 管理方針

- 各写真は、**層ごとの作業プロセス確認と考察**に活用します。
- ファイル名には必ず**撮影日（YYYY-MM-DD 形式）**を付け、作業時系列と対応させます。
- 畝構造変更や追加観察が生じた場合も、**原則として新たに日付付きで追加保存**します。
